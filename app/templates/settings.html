Google OAuth Configuration

To use Google Photos, you need to set up OAuth credentials:

  1. Create a project in Google Cloud Console
  2. Enable Google Photos Library API
  3. Create OAuth 2.0 credentials (Web application)
  4. Add redirect URI: {{ request.url.scheme }}://{{ request.url.netloc }}/auth/google/callback
  5. Set environment variables: GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET
